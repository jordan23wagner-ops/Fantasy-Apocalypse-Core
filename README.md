# Fantasy-Apocalypse-Core

**Genre:** Diablo-style ARPG meets Fallout-style exploration RPG.
**Setting:** Fantasy Apocalypse. A world where magic and high-fantasy elements have been ravaged by a cataclysmic event.

### Core Pillars:
- **Base of Operations:** Players start and return to a central Town/Base/Camp.
- **The Loop:** Explore the wasteland → Loot gear/secrets → Level up skills → Hunt Bosses.
- **Mixed Pacing:**
    - **Hack n' Slash Zones:** High-density areas focused on fast-paced combat and "loot pinatas" (Diablo-style).
    - **Exploration Zones:** Slower-paced areas focused on hunting specific bosses, finding secrets, and environmental storytelling (Fallout-style).
- **Loot System:** Deep, exciting itemization is the primary driver for character progression.
